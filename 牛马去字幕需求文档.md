# 牛马去字幕小程序 - 复刻需求文档

## 📱 项目概述

**目标**: 复刻「牛马去字幕」微信小程序 - AI 智能视频去字幕工具

**核心价值**: 用户上传视频，AI 自动识别并去除字幕/水印/文字，输出干净视频

## 🔌 技术架构

### 前端技术栈
```
- uni-app (Vue3 + Vite + TypeScript)
- Pinia 状态管理
- uni.request 网络请求
- uni.navigateTo 路由
- wx.createVideoContext 视频播放
- wx.chooseMedia 选择视频
- wx.saveVideoToPhotosAlbum 保存视频
```

### 后端技术栈
```
- Node.js + Express / Python + Flask
- MySQL 数据库
- Redis 缓存
- 阿里云 OSS 对象存储
- 阿里云视频处理 API（或类似第三方 API）
- WebSocket 实时推送处理进度
```

### 核心 API 接口

#### 视频处理 API（第三方）
```
方案 1: 阿里云视频 SAM
- API: 视频文字擦除
- 文档：https://help.aliyun.com/product/89777.html

方案 2: 腾讯云智能视频处理
- API: 视频去字幕
- 文档：https://cloud.tencent.com/product/smp

方案 3: 自建 FFmpeg + AI 模型
- 使用 OpenCV + PyTorch
- WASD 目标检测
- 视频帧处理
```

CREATE TABLE users (

  id INT PRIMARY KEY AUTO_INCREMENT,

## ⚠️ 注意事项

1. **合规性**
   - 用户协议和隐私政策必须完善
   - 版权提示：用户需保证上传视频有合法使用权
   - 内容安全检测：禁止处理违法违规内容

2. **性能优化**
   - 视频 Compression 后再上传
   - CDN 加速视频下载
   - 分页加载历史记录
   - 图片懒加载

3. **用户体验**
   - 处理进度实时反馈
   - 失败任务自动重试
   - 处理完成微信模板消息通知
   - 新手引导教程

4. **成本控制**
   - 阿里云 API 调用成本核算
   - 设置合理的积分消耗
   - 免费用户体验次数限制
