## Documentation Maintenance

### How to Update Documentation

1. Fork the project
2. Edit the corresponding document
3. Follow [Documentation Standards](./documentation-standards.md)
4. Submit a PR
5. Wait for review and merge

### Documentation Standards

- Follow [documentation-standards.md](./documentation-standards.md) for all writing
- Use templates from `docs/templates/` for new documents
- Run `docs/scripts/validate-docs.sh` before submitting
- Keep documents up to date with code changes

### Feedback

If you find documentation issues or have improvement suggestions:

- Submit an [Issue](https://github.com/sypw233/video-subtitle-remover/issues)
- Or submit a PR with the fix
