# 微信小程序审核指南

**项目名称**: 视频去字幕小程序
**编写日期**: 2026-06-13
**适用平台**: 微信小程序

---

## 一、微信小程序审核要点

### 1.1 基础要求

#### 信息完整性
- **小程序名称**: 必须与实际功能相关，不能包含"微信"、"测试"等禁止词汇
- **小程序简介**: 必须清晰描述功能，不能夸大或虚假宣传
- **小程序头像**: 必须清晰，不能包含二维码
- **服务类目**: 必须选择与功能匹配的类目（推荐：工具 > 视频工具 或 效率 > 文件管理）

#### 服务类目选择建议
```
主类目: 工具
子类目: 视频工具 / 信息查询
备选类目: 效率 > 文件管理
```

**注意**: 部分类目需要特殊资质：
- 如涉及用户付费，可能需要"生活服务 > 生活缴费"类目或 ICP 证
- 如涉及视频编辑，可能需要相关资质

### 1.2 功能要求

#### 功能完整性
- 所有页面必须可正常访问
- 所有按钮功能必须可用
- 不能出现空白页面或占位内容
- 不能有无法完成的操作流程

#### 功能稳定性
- 不能出现频繁崩溃
- 网络请求超时需有提示
- 加载状态需有 loading 提示
- 错误状态需有友好提示

### 1.3 设计要求

#### UI 规范
- 符合微信小程序设计规范
- 页面层级不超过 10 层
- tabBar 最少 2 个、最多 5 个
- 不能有外部链接或二维码

#### 图标规范
- TabBar 图标尺寸: 81x81 px
- 图标格式: PNG（不支持 SVG）
- 图标文件大小: < 40KB
- 需提供选中和未选中两种状态

### 1.4 内容要求

#### 内容安全
- 不能有违法违规内容
- 不能有色情、暴力、赌博内容
- 不能有虚假广告
- 用户生成内容需要审核机制

#### 知识产权
- 不能侵犯他人知识产权
- 使用的图片、字体需有授权
- 名称不能与知名品牌混淆

## 二、常见驳回原因及解决方案

### 2.1 功能类驳回

| 驳回原因 | 频率 | 解决方案 |
|----------|------|----------|
| **功能不完整** | 高 | 确保所有页面文件存在且功能可运行。当前项目有 7 个页面缺失，必须补全 |
| **页面空白** | 高 | 确保每个页面有实际内容，不能只有 loading 或空白 |
| **功能不可用** | 高 | 测试所有功能流程，确保按钮点击有响应 |
| **诱导分享** | 中 | 不能强制用户分享才能使用功能 |
| **虚拟支付** | 高 | iOS 端不能使用非 IAP 方式购买虚拟商品 |

### 2.2 配置类驳回

| 驳回原因 | 频率 | 解决方案 |
|----------|------|----------|
| **AppID 为空** | 高 | 在 `manifest.json` 中填写正确的 AppID |
| **域名未配置** | 高 | 在微信后台配置合法域名，`urlCheck` 设为 `true` |
| **权限未声明** | 高 | 在配置中声明所有使用的权限 |
| **隐私协议缺失** | 高 | 配置 `__usePrivacyCheck__` 并实现隐私弹窗 |
| **服务类目不符** | 中 | 选择与功能匹配的服务类目 |

### 2.3 内容类驳回

| 驳回原因 | 频率 | 解决方案 |
|----------|------|----------|
| **缺乏内容审核** | 高 | 接入微信内容安全 API (`security.msgSecCheck`) |
| **UGC 无审核** | 高 | 用户上传视频需进行内容审核 |
| **版权风险** | 中 | 去字幕功能可能涉及版权问题，需在用户协议中声明使用限制 |
| **虚假宣传** | 中 | 功能描述必须与实际功能一致 |

### 2.4 技术类驳回

| 驳回原因 | 频率 | 解决方案 |
|----------|------|----------|
| **包体过大** | 中 | 主包不超过 2MB，使用分包加载 |
| **加载过慢** | 中 | 首屏加载时间控制在 3 秒内 |
| **频繁崩溃** | 低 | 充分测试，处理异常情况 |
| **API 不兼容** | 中 | 确保不使用小程序不支持的 Web API |

## 三、优化建议

### 3.1 性能优化

#### 首屏加载优化
```
1. 使用分包加载，将非首屏页面放入分包
2. 图片资源使用 CDN，避免打包进小程序
3. 使用懒加载延迟加载非关键资源
4. 减少首屏渲染的 setData 数据量
```

#### 包体积优化
```
1. 移除未使用的代码和资源
2. 图片压缩（TinyPNG 等工具）
3. 使用分包预加载
4. 清理 macOS 资源文件 (._*)
5. 移除开发环境代码（console.log 等）
```

#### 网络优化
```
1. 使用缓存减少重复请求
2. 合并小请求减少网络开销
3. 使用 WebSocket 替代轮询（如果后端支持）
4. 请求超时设置合理值（30s）
5. 弱网环境下有降级方案
```

### 3.2 体验优化

#### 加载状态
```vue
<!-- 示例: 全局 loading 组件 -->
<template>
  <view v-if="loading" class="loading-mask">
    <view class="loading-spinner"></view>
    <text class="loading-text">{{ loadingText }}</text>
  </view>
</template>
```

#### 错误状态
```vue
<!-- 示例: 网络错误状态 -->
<template>
  <view v-if="error" class="error-state">
    <image src="/static/error.png" class="error-image" />
    <text class="error-text">{{ errorMessage }}</text>
    <button @tap="retry">重试</button>
  </view>
</template>
```

#### 空状态
```vue
<!-- 示例: 空列表状态 -->
<template>
  <view v-if="!loading && list.length === 0" class="empty-state">
    <image src="/static/empty.png" class="empty-image" />
    <text class="empty-text">暂无数据</text>
    <button @tap="goCreate">去创建</button>
  </view>
</template>
```

### 3.3 安全优化

#### 内容审核
```typescript
// 后端: 调用微信内容安全 API
async function checkContent(content: string): Promise<boolean> {
  const response = await axios.post(
    'https://api.weixin.qq.com/wxa/msg_sec_check',
    {
      content,
      version: 2,
      scene: 2, // 2=视频
      openid: userOpenid
    },
    {
      params: { access_token: await getAccessToken() }
    }
  );
  return response.data.errcode === 0;
}
```

#### 文件安全
```typescript
// 验证上传文件类型
function validateFileType(file: File): boolean {
  const allowedTypes = ['video/mp4', 'video/quicktime', 'video/x-msvideo'];
  return allowedTypes.includes(file.type);
}

// 验证文件大小
function validateFileSize(size: number, maxSize: number): boolean {
  return size <= maxSize * 1024 * 1024;
}
```

## 四、提审清单

### 4.1 提审前自检

#### 配置检查
- [ ] `manifest.json` 中 `appid` 已填写
- [ ] `manifest.json` 中 `name` 已填写（不超过 30 字符）
- [ ] `manifest.json` 中 `description` 已填写（不超过 120 字符）
- [ ] `urlCheck` 设为 `true`
- [ ] 隐私协议已配置（`__usePrivacyCheck__: true`）
- [ ] 权限声明完整（`permission`）
- [ ] 隐私信息声明完整（`requiredPrivateInfos`）
- [ ] 服务类目已选择
- [ ] 服务器域名已在微信后台配置

#### 页面检查
- [ ] 所有页面文件存在
- [ ] 所有页面可正常访问
- [ ] TabBar 图标文件存在且正确
- [ ] 无空白页面
- [ ] 页面标题正确

#### 功能检查
- [ ] 登录流程正常
- [ ] 视频选择正常
- [ ] 视频上传正常
- [ ] 任务创建正常
- [ ] 任务轮询正常
- [ ] 结果预览正常
- [ ] 结果下载正常
- [ ] 历史列表正常
- [ ] 个人中心正常
- [ ] 充值支付正常（如有）

#### 安全检查
- [ ] 内容审核 API 已接入
- [ ] XSS 防护已实现
- [ ] 敏感信息未暴露
- [ ] Token 安全存储
- [ ] 接口鉴权正常

#### 体验检查
- [ ] Loading 状态完整
- [ ] Error 状态友好
- [ ] 空状态有引导
- [ ] 弱网有降级
- [ ] 操作有反馈

### 4.2 测试用例

#### 功能测试
```
1. 首页
   - 点击"去除字幕"按钮 → 跳转上传页
   - 点击"去除图标"按钮 → 跳转上传页
   - 已登录时显示统计数据
   - 未登录时隐藏统计数据

2. 上传页
   - 点击选择视频 → 打开相册/相机
   - 选择视频后显示预览
   - 视频超过 100MB 显示错误提示
   - 点击开始上传 → 显示进度条
   - 上传过程中可暂停/继续
   - 上传完成后跳转预览页

3. 预览页
   - 显示视频预览
   - 显示处理选项
   - 点击开始处理 → 创建任务

4. 处理页
   - 显示处理进度
   - 轮询任务状态
   - 处理完成后自动跳转

5. 结果页
   - 显示处理结果
   - 可预览视频
   - 可下载视频
   - 可分享结果

6. 历史页
   - 显示任务列表
   - 下拉刷新
   - 上拉加载更多
   - 点击任务可查看详情

7. 个人中心
   - 显示用户信息
   - 显示积分余额
   - 显示处理次数
   - 点击充值跳转充值页

8. 登录
   - 微信授权登录
   - 登录成功返回上一页
   - 登录失败显示错误

9. 充值
   - 选择充值金额
   - 调起支付
   - 支付成功更新余额
   - 支付失败显示错误
```

#### 异常测试
```
1. 网络异常
   - 断开网络 → 显示网络错误
   - 弱网环境 → 显示 loading
   - 请求超时 → 显示超时提示

2. 权限异常
   - 拒绝相册权限 → 提示开启权限
   - 拒绝相机权限 → 提示开启权限

3. 文件异常
   - 选择非视频文件 → 显示错误
   - 选择超大文件 → 显示错误
   - 上传中断 → 可断点续传

4. 状态异常
   - Token 过期 → 跳转登录
   - 任务失败 → 显示错误和重试
   - 并发操作 → 不重复提交
```

### 4.3 提审材料

#### 必需材料
1. **小程序二维码** - 用于审核人员体验
2. **测试账号** - 如需登录，提供测试账号和密码
3. **功能说明文档** - 说明核心功能和使用流程
4. **隐私协议** - 隐私政策页面链接
5. **服务条款** - 用户服务协议页面链接

#### 可选材料
1. **特殊资质** - 如涉及特殊类目
2. **版权授权** - 如使用第三方素材
3. **安全审计报告** - 如有第三方安全审计

### 4.4 审核时间

- **首次提审**: 通常 1-7 个工作日
- **加急审核**: 可申请加急（需满足条件）
- **审核结果**: 通过微信通知或邮件

### 4.5 审核被拒后处理

1. **查看驳回原因** - 在微信公众平台查看详细原因
2. **修改代码** - 根据原因修改代码和配置
3. **重新提审** - 修改完成后重新提交
4. **申诉** - 如认为误判，可提交申诉
