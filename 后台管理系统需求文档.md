# 后台管理系统需求文档

## 📋 系统概述

**目标**: 为「牛马去字幕」小程序提供完整的运营管理后台

**技术栈**: 
- 前端: Vue3 + Element Plus（或 React + Ant Design）
- 后端: Node.js + Express（与小程序共用后端）
- 部署: 独立域名，管理员登录

---

## 🎯 功能模块

### 1. 数据概览（Dashboard）

| 指标 | 说明 |
|------|------|
| 今日新增用户 | 每日新注册用户数 |
| 今日活跃用户 | 每日登录用户数 |
| 今日任务数 | 每日处理任务总数 |
| 今日收入 | 每日充值金额 |
| 任务成功率 | 处理成功/失败比例 |
| 用户留存率 | 1日/7日/30日留存 |
| 积分消耗统计 | 每日积分消耗总量 |
| 在线用户数 | 当前在线用户数 |

**展示形式**:
- 数字卡片（今日数据）
- 折线图（7天/30天趋势）
- 饼图（任务类型分布）
- 柱状图（收入趋势）

## 🔧 技术方案

### 前端技术栈
```
- Vue3 + Composition API
- Element Plus UI 组件库
- Vue Router 路由管理
- Pinia 状态管理
- Axios 网络请求
- ECharts 图表
- Excel 导出（xlsx.js）
```

### 后端技术栈
```
- Node.js + Express（与小程序共用）
- MySQL 数据库（共用）
- Redis 缓存（共用）
- JWT 认证
- RBAC 权限控制
- Multer 文件上传
```

### 部署方案
```
- 前端：Nginx 静态部署
- 后端：PM2 进程管理
- 域名：admin.example.com
- SSL：Let's Encrypt 免费证书
```

## 📊 数据库新增表

```sql
-- 管理员表
CREATE TABLE admins (
    id VARCHAR(36) PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    password VARCHAR(100) NOT NULL,
    nickname VARCHAR(50),
    role_id VARCHAR(36),
    is_active BOOLEAN DEFAULT TRUE,
    last_login_at DATETIME,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 角色表
CREATE TABLE roles (
    id VARCHAR(36) PRIMARY KEY,
    name VARCHAR(50) UNIQUE NOT NULL,
    description VARCHAR(200),
    permissions JSON COMMENT '权限列表',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 操作日志表
CREATE TABLE admin_logs (
    id VARCHAR(36) PRIMARY KEY,
    admin_id VARCHAR(36) NOT NULL,
    action VARCHAR(50) NOT NULL,
    target_type VARCHAR(50),
    target_id VARCHAR(36),
    detail JSON,
    ip_address VARCHAR(50),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_admin_id (admin_id),
    INDEX idx_action (action),
    INDEX idx_created_at (created_at)
);

-- 系统配置表
CREATE TABLE system_configs (
    id VARCHAR(36) PRIMARY KEY,
    config_key VARCHAR(50) UNIQUE NOT NULL,
    config_value TEXT NOT NULL,
    description VARCHAR(200),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

-- 公告表
CREATE TABLE announcements (
    id VARCHAR(36) PRIMARY KEY,
    title VARCHAR(100) NOT NULL,
    content TEXT NOT NULL,
    type ENUM('notice', 'activity', 'update') DEFAULT 'notice',
    is_active BOOLEAN DEFAULT TRUE,
    start_time DATETIME,
    end_time DATETIME,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- FAQ 表
CREATE TABLE faqs (
    id VARCHAR(36) PRIMARY KEY,
    question VARCHAR(200) NOT NULL,
    answer TEXT NOT NULL,
    category VARCHAR(50),
    sort_order INT DEFAULT 0,
    is_active BOOLEAN DEFAULT TRUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```
